# LinkedIn Easy Apply Automation Test
This project automates the LinkedIn Easy Apply flow using Selenium and TestNG.
## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
## Project Structure
- pages → Page Object classes
- tests → Test cases
- base → WebDriver setup
## Run Test

Set environment variables:

Mac/Linux:
export LINKEDIN_EMAIL=your_email
export LINKEDIN_PASSWORD=your_password

Run:
mvn clean test
